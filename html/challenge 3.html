All About Headings

In an HTML document, headings act like titles. These titles come in various sizes, much like the ones we find when we do word processing in Microsoft Word/Google Docs.

It is important to remember, headings are for important titles. Don't use headings to represent bold or larger front sized words. There are other tags that we use to perform those functions. More on that in Challenge 2.
